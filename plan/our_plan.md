# Project Plan: Data Analyst Portfolio Website for Bill Nattawut

## 1. Project Overview
สร้างเว็บไซต์ Portfolio ส่วนตัวสำหรับสายงาน Data Analyst เพื่อแสดงทักษะและผลงาน โดยเน้นความทันสมัย ใช้งานง่าย และรองรับการแสดงผลบน GitHub Pages

## 2. Technical Specifications
- **Theme:** Modern Dark Theme (พื้นหลังโทนเข้ม, ข้อความสีสว่าง)
- **Typography:** ฟอนต์ **Sarabun** (ผ่าน Google Fonts)
- **Tech Stack:** HTML5, CSS3 (Vanilla CSS), JavaScript
- **Hosting:** GitHub Pages

## 3. Website Structure (Sections)
### 1. Hero Section
- **Heading:** Bill Nattawut
- **Sub-heading:** Data Analyst
- **Description:** ประวัติย่อหรือเป้าหมายในการทำงานด้านข้อมูล

### 2. Projects Section (3 Projects)
จะแสดงผลในรูปแบบ Cards พร้อมรายละเอียดดังนี้:
1. **Project 1:** ชื่อโปรเจกต์, คำอธิบายสั้นๆ, ทักษะที่ใช้ (เช่น Python, SQL), ลิงก์ไปที่ GitHub
2. **Project 2:** ชื่อโปรเจกต์, คำอธิบายสั้นๆ, ทักษะที่ใช้, ลิงก์ไปที่ GitHub
3. **Project 3:** ชื่อโปรเจกต์, คำอธิบายสั้นๆ, ทักษะที่ใช้, ลิงก์ไปที่ GitHub

### 3. Contact Details
- ข้อมูลการติดต่อ (Email, LinkedIn, GitHub)
- ลิงก์ Social Media (ถ้ามี)

## 4. Step-by-Step Implementation Plan

### Phase 1: Setup & Design
1. สร้างโครงสร้างโฟลเดอร์โปรเจกต์
2. สร้างไฟล์พื้นฐาน: `index.html`, `style.css`, `script.js`
3. เชื่อมต่อ Google Fonts สำหรับฟอนต์ Sarabun
4. กำหนด CSS Variables สำหรับ Dark Theme (Colors, Spacing)

### Phase 2: HTML/CSS Development
1. เขียน HTML Markup สำหรับ Hero, Projects และ Contact
2. ตกแต่ง Layout ให้เป็น Responsive (รองรับมือถือและคอมพิวเตอร์)
3. ใส่ข้อมูล 3 โปรเจกต์ที่เตรียมไว้

### Phase 3: Git & Deployment
1. สร้าง Repository บน GitHub
2. Push โค้ดขึ้นไปที่ Branch `main`
3. ตั้งค่า GitHub Pages ในเมนู Settings ของ Repository
4. ตรวจสอบความถูกต้องของลิงก์และหน้าจอแสดงผล
